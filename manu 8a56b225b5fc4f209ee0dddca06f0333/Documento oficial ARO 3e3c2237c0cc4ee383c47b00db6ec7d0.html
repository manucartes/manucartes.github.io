Python

Python es un lenguaje de programación interpretado de alto nivel con una filosofía de diseño que enfatiza la legibilidad del código. Es conocido por su sintaxis simple y fácil de aprender, que mejora la legibilidad del código y reduce el costo de mantenimiento del programa. Python permite a los programadores expresar conceptos en menos líneas de código que otros lenguajes como C++ o Java.

Función de salida

Para mostrar información hacia el usuario, podemos utilizar la función print()

print("Mi mensaje")

Este código muestra la frase Mi mensaje. Notar que cuando quiero mostrar texto debe ir entre comillas.

Concatenar

Unir texto con variables. Para ello se utiliza la ==coma==.

nombre = "Raul"
estatura = 1.7
edad = 27
print("Mi nombre es",nombre, "mi estatura es",estatura, "[m] y tengo", edad,"años")

Variables

Son palabras o frases, sin espacios ni símbolos que permiten guardar datos en su interior. Fisicamente, estos datos quedan almacenados en memoria RAM.

Los datos que puedo almacenar son:

  • Variables de texto: nombre = "manu"
  • Variables de numeros enteros: edad = 21
  • Variables de numeros decimales: estatura = 1.7

Función de entrada

Permite que el usuario ingrese valores dentro de las variables.

variable = input("Ingrese su nombre:")

IMPORTANTE

La función input() procesa toda la información como si se tratase de texto.

Si queremos trabajar con números computables matemáticamente, debemos usar una de estas funciones:

  • int(): Convierte a numero entero.
  • float(): Convierte a numero decimal

Ejemplo

nacimiento = int(input("Ingrese año de nacimiento:"))
edad = 2023 - nacimiento
print("Su edad es", edad, "años")

Condición if

Esta condicion permite que dentro de nuestro algoritmo sea posible tomar una decisión.

Estas decisiones son del tipo Si sale el sol entonces habrá calor, Si el numero es mayor a 5 entonces el programa se volverá a repetir.

SINTAXIS

if COMPARACION:  
	#CODIGO SI LA COMPARACION SE CUMPLE

	#CODIGO SI LA COMPARACION NO SE CUMPLE

Ciclo for

  • Uno de los usos es generar números entre un rango.
  • El ciclo for tambien se utiliza para recorrer listas.

Generando números dentro de un rango

SINTAXIS

for i in range(7):  
	#AQUI ESCRIBO MI CODIGO  
	#PUEDEN SER VARIAS LINEAS DE CODIGO
	print("Hola!")

Modos de uso

  • range(21): Cuenta desde 0 hasta 20. El código del interior se ejecuta 20 veces en este caso.
  • range(5,21): Cuenta desde 5 hasta 20. El código del interior se ejecuta 15 veces en este caso.
  • range(10, 21, 2): Cuenta desde 10 hasta 20 con incrementos de 2. El código del interior se ejecuta 5 veces en este caso.

Métodos de listas

  • .append(valor): Añade valor a la lista.
  • .pop(indice): Borra un elemento en el indice especificado.
  • .remove(valor): Busca a valor y lo elimina. Solo elimina al primer valor que encuentra.
  • .sort(): Ordena la lista de menor a mayor.
  • .sort(reverse=True): Ordena la lista de mayor a menor. La T de True debe ir en mayúscula.

¿Como calcular si un numero es par o impar?

Se debe calcular su resto con el operador %.

Ejemplo 1:

num = 4
res = num %2
print(res)

En este ejemplo el resultado es 0 debido a que el resto de $\frac{4}{2}$ es 0.

Ejemplo 2:

num = 5
res = num %2
print(res)

En este ejemplo el resultado es 1 debido a que el resto de 52\frac{5}{2}25​﻿ es 1.

¿Y si queremos decirle al usuario si es par o impar con palabras?

Para esto solo necesitas una condicion if.

Ejemplo

num = 4
res = num %2
print(res)
if res == 0:  
	print("El numero es par")
else:  
	print("El numero es impar")
