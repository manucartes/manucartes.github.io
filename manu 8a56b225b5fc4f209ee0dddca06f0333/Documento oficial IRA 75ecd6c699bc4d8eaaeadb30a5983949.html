¿Que es Python?

Python es un lenguaje de programación interpretado, de alto nivel y de propósito general. Creado por Guido van Rossum y lanzado por primera vez en 1991, Python tiene un diseño filosófico que enfatiza la legibilidad del código, y una sintaxis que permite a los programadores expresar conceptos en menos líneas de código de lo que sería posible en lenguajes como C++ o Java. Se utiliza para una amplia variedad de aplicaciones, incluyendo desarrollo web, análisis de datos, aprendizaje automático, inteligencia artificial y desarrollo de software.

Función de salida

Para mostrar información hacia el usuario, podemos utilizar la función print()

print("Mi mensaje")

Este código muestra la frase Mi mensaje. Notar que cuando quiero mostrar texto debe ir entre comillas.

Concatenar

Unir texto con variables. Para ello se utiliza la ==coma==.

nombre = "Raul"
estatura = 1.7
edad = 27

print("Mi nombre es",nombre, "mi estatura es",estatura, "[m] y tengo", edad,"años")

Variables

Son palabras o frases, sin espacios ni símbolos que permiten guardar datos en su interior. Fisicamente, estos datos quedan almacenados en memoria RAM.

Los datos que puedo almacenar son:

  • Variables de texto: nombre = "manu"
  • Variables de numeros enteros: edad = 21
  • Variables de numeros decimales: estatura = 1.7

Función de entrada

Se utiliza para que el usuario interactue con las variables del programa. Esta función corresponde a input().

Ejemplo

edad = input("Ingrese su edad:")
print("Su edad es", edad)

Notar que dentro del input() podemos escribir un texto para indicarle al usuario que dato debe escribir.

⚠️ IMPORTANTE

La función input() procesa todo lo que es escriba (incluso números) como si se tratase de texto.

Si queremos trabajar con aquellos números para hacer operaciones matemáticas debemos convertirlos con una de las siguientes funciones.

  • int(): Convierte a número entero.
  • float(): Convierte a número decimal.

Ejemplo

a = int(input("Ingrese a:"))
b = int(input("Ingrese b:"))

suma= a + b

print("La suma es", suma)

Si quisiéramos trabajar con decimales debe utilizar la función float()

Ciclo for

Sirve para:

  • Generar numeros entre un rango.
  • Reptir código una determinada cantidad de veces.
  • Recorrer listas.

SINTAXIS: Generando numeros entre un rango.

for i range(8):
  #CODIGO
  #CODIGO
  #CODIGO

Formas de usar range()

  • range(21): Genera numeros de 0 a 20.
  • range(8,21): Genera numeros desde 8 a 20.
  • range(8,21,2): Genera numeros entre 8 y 20 con incrementos de 2.
